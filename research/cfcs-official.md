# Communication Function Classification System (CFCS) Official Resource

[CFCS Official Website](https://cfcs.us/)

## Key Findings

### CFCS Purpose

> "The Communication Function Classification System (CFCS) was created and validated for children with cerebral palsy that can be used by a wide variety of individuals who are interested in cerebral palsy."

### Development

Reference for the initial CFCS Development: Hidecker, M.J.C., Paneth, N., Rosenbaum, P.L., Kent, R.D., Lillie, J., Eulenberg, J.B., Chester, K., Johnson, B., Michalsen, L., Evatt, M., & Taylor, K.
