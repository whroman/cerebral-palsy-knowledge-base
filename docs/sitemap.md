### Classification Systems

Doctors use these scales to describe how cerebral palsy affects daily function:

- [Classification Overview](./understanding/index.md) — What classification means and why it matters
- [GMFCS](./understanding/gmfcs.md) — Gross motor function (mobility and movement)
- [MACS](./understanding/macs.md) — Manual ability (using hands and arms)
- [CFCS](./understanding/cfcs.md) — Communication function (sending and receiving messages)
